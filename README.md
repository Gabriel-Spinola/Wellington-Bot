# Wellington Bot
Bot for sending messages from a telegram chat to a discord chat

## Technologies Used
- Node.js: The runtime environment for executing the bot's JavaScript code.
- Discord.js: Interacting with the Discord API.
- Telegram API: Enables communication with the Telegram platform.
- Fly.io: Used for deployment.
- Ngrok: Used for deploying a test env.
